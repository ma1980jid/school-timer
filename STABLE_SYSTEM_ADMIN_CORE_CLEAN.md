# Stable Release: System Admin Core Clean

## اسم النسخة
stable-system-admin-core-clean

## سبب التثبيت
تم تثبيت هذه النسخة بعد تنظيف لوحة مدير النظام من المصدر مباشرة، وإزالة عناصر التصاميم المتعددة التي كانت تسبب ثقلًا أو تعليقًا في بعض المتصفحات.

## ما تم اعتماده
- لوحة مدير النظام تفتح بدون تعليق.
- إزالة عناصر التصاميم المتعددة من `system-admin.html`.
- إزالة منطق التصاميم والمناسبات من `system-admin.js`.
- تبسيط لوحة مدير النظام لتكون مخصصة لإدارة المدارس فقط.
- إبقاء التصميم العام الواحد ثابتًا لكل المدارس.

## الوظائف المعتمدة في لوحة مدير النظام
- إضافة مدرسة.
- تعديل بيانات مدرسة.
- إدخال الرابط المختصر `school_slug`.
- تحديد المحافظة والولاية.
- إدخال رمز مدير المدرسة.
- تفعيل / إيقاف المدرسة.
- إدخال رابط شعار المدرسة أو رفع الشعار.
- عرض روابط المدرسة.
- نسخ روابط المدرسة.
