## الهدف من هذه النسخة

تثبيت نسخة مستقرة بعد نجاح اختبارات تعدد المدارس، واستقلال بيانات كل مدرسة، وإصلاح مشاكل الكاش في المتصفح والتطبيق، وإضافة حذف المدرسة من لوحة مدير النظام.

## حالة النسخة

- شاشة العرض للهاتف والحاسوب تعمل عبر school_slug.
- كل مدرسة لها اسم وشعار ورسائل وإعلانات مجدولة مستقلة.
- تم منع ظهور بيانات مدرسة الشيخ سيف في المدارس الأخرى.
- تم إصلاح روابط لوحة مدير المدرسة.
- تم تحسين سلوك Safari وPWA.
- تم إضافة حذف المدرسة مع التحقق الفعلي من قاعدة البيانات.
