# Stable Release: Dashboard QR Actions Layout

## اسم النسخة
stable-dashboard-qr-actions-layout

## سبب التثبيت
تم تثبيت هذه النسخة بعد نجاح تعديلات QR وتنظيم أزرار لوحة مدير المدرسة ولوحة مدير النظام.

## ما تم اعتماده
- إضافة QR Code لروابط لوحة مدير النظام.
- إضافة QR Code لروابط لوحة مدير المدرسة.
- QR يتغير تلقائيًا حسب `school_slug` لكل مدرسة.
- إضافة زر نسخ صورة QR بدل نسخ رابط QR.
- تقصير عنوان QR الحاسوب في لوحة مدير المدرسة إلى: `QR الحاسوب`.
- تنظيم أزرار لوحة مدير المدرسة لتخفيف ازدحام الجانب الأيسر.
- حذف زر دليل المستخدم المكرر من أوامر إضافية مع بقاء زر دليل المستخدم الأساسي.
- إبقاء زر الإعلانات المجدولة في الجانب الأيسر وعدم نقله إلى الجانب الأيمن.
- منع ازدحام الجانب الأيمن حتى لا تدفع العناصر للأسفل.
